# Invoke-Pre2kSpray

Modified DomainPasswordSpray version to enumerate machine accounts and perform a pre2k password spray.

### Usage 

```powershell
Invoke-Pre2kSpray -OutFile valid-creds.txt -Domain test.local -Force
```
